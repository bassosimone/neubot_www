A new version is available

Please, note that an updated version of Neubot is available at Neubot.org.

Since version 0.4.11, Neubot includes support for automatic updates in the Windows environment. However, as the digital signature system for the updates is not yet implemented, this piece of work is still marked as "experimental". Hence, the feature is currently disabled, and shall stay so until there is sign/verify code. Meanwhile, should you want to exploit the feature, simply set "win32_updater" to "1", using the settings page.
